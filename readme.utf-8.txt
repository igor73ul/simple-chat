ОПИСАНИЕ:
 Упрощённая реализация чата в одном процессе.
 В основе клиент-серверный подход.
 Основные возможности: подключение/отключение, передача сообщений, 
 контроль состояния подключения
СБОРКА/УСТАНОВКА:
 Для сборки используется СMake, Qt5. Может потребоваться указать путь
 к Qt через CMAKE_PREFIX_PATH в главном файле CMakeList.txt (в корне), 
 если cmake не сможет автоматически найти эти библиотеки.
 src -- исходный код программы. 
 config -- настроечный файл.
 include -- будут складываться по мере сборки заголовочные файлы.
 lib -- будут складываться собранные библиотеки.
 bin -- исполняемый файл.
ЗАПУСК:
 в папке bin исполняемый файл - mainservice. Открывшийся виджет управляет
 всем приложением: запуск/остановка чат-сервера, создание новых клиентов,
 останавливает работу всего приложения.
 Каждому новому клиенту необходимо задать имя. В случаи потери связи 
 с сервером есть возможность попробовать восстановить подключение. 
 Каждый клиент сам отвечает за своё "время жизни". Только по своей инициативе
 отключается и уничтожает себя.
